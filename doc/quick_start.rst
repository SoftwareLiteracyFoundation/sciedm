###############
Getting started
###############


scikit-learn compatible EDM
===========================

Download skedm
--------------

Clone the ``skedm`` repository:

.. prompt:: bash $

  git clone https://github.com/SoftwareLiteracyFoundation/skedm

Install `skedm`:

.. prompt:: bash $

  cd skedm
  python -m pip install .
